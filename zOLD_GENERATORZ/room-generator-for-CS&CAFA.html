CS & CAFA Room Inventory

This tool generates a CSV file containing the data extracted from:

  • CS Rooms Inventory 2024-2025 (Federizo Hall 1st/2nd Floor & Labs)
  • CAFA Facilities Inventory 2025 (Federizo Hall 3rd Floor)

It includes the inferred columns Has_Whiteboard and Has_TV based on the room classification.

Data Logic Used:
- Aircon: Set to 'Yes'/'No' for CAFA (as specified in doc); 'Unknown' for CS.
- Whiteboard: Assumed 'Yes' for Lecture Rooms, Labs, and Drafting Rooms.
- TV: Assumed 'Yes' for AVRs, Multimedia Labs, and Computer Rooms only.
